The back end and the front end run on different servers you need to install expo in order to make the front server works,
For the back end you will need to create a virtual environment with conda for exemple because it require TensorFlow 1
In order to test the model you will have to extract the training_dataset.zip then you will have to run the following command in order to train the model: bash train.sh
if you want to test the algorithme you will have to run the following command: python3 testing.py
